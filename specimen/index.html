    Program K-Means Clustering¶

    Menerapkan K-Means Clustering Pada Data Nilai Mahasiswa¶

    Langkah pertama : Import Libraries¶

    import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.prepocessing import MinMaxScaler

    Seperti yang anda lihat di atas, kita mengimport libraries :

    Pandas untuk membaca dan menulis Spreadsheet

    Numpy untuk melakukan perhitungan yang efisien

    Matplotlib untuk visualisasi data

    Langkah kedua : Menginput Data¶

    Data yang digunakan yakni data nilai mahasiswa 

    mhs = pd.read_csv("nilaimhs2.csv")
mhs.head()

    Langkah ketiga¶

    menentukan variabel yang akan di clusterkan

    mhs_x = mhs.iloc[:, 1:3]
mhs_x.head()

    kemudian di visualisasikan persebaran datanya.

    plt.scatter(mhs.UTS, mhs.UAS, s =10, c = "c", marker = "o", alpha = 1)
plt.show()

    Langkah Keempat : Menentukan Nilai K¶

    x_array =  np.array(driver_x)
print(x_array)
    scaler = MinMaxScaler()
x_scaled = scaler.fit_transform(x_array)
x_scaled
    --- Menentukan dan mengkonfigurasi fungsi kmeans ---

kmeans = KMeans(n_clusters = 3, random_state=123)

--- Menentukan kluster dari data ---

kmeans.fit(x_scaled)

    Langkah Kelima : Memvisualkan Cluster¶

    print(kmeans.cluster_centers_)

    --- Menampilkan Hasil Kluster ---

print(kmeans.labels_)

--- Menambahkan Kolom "kluster" Dalam Data Frame Mhs ---

driver["kluster"] = kmeans.labels_

    Kemudian memvisualisasikan hasil cluster

    --- Memvisualkan hasil kluster ---

output = plt.scatter(x_scaled[:,0], x_scaled[:,1], s = 100, c = mhs.kluster, marker = "o", alpha = 1, )

centers = kmeans.cluster_centers_
plt.scatter(centers[:,0], centers[:,1], c='red', s=200, alpha=1 , marker="s");
plt.title("Hasil Klustering K-Means")
plt.colorbar (output)
plt.show()

    Dari gambar diatas dapat kita lihat bahwa dari data nilai mahasiswa telah ter cluster menjadi 3 .
