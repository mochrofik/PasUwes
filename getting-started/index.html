    Hal yang perlu di install¶

    Sebelum mengerjakan program K-Means Clustering menggunakan bahasa python anda harus install library tertentu terlebih dahulu pada command prompt python.

    1. Pandas

    pip install pandas

    1. Numpy

    pip install numpy

    1. Matplotlib

    pip install matplotlib
